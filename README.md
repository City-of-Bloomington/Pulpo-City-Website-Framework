# City Website Framework

Building a good city website is surprisingly difficult. This is a shared space to track best practices and processes for the task. 

## General steps:

  - Foundation
    - Design Manual
    - Writing Style Guide
    - Vision and Goals
    - Information Architecture
    - User Stories
  - Assembly 
    - Wireframes
    - Page Types
  - Implementation
    - Rewrite Content
    - Redesign CSS / HTML
